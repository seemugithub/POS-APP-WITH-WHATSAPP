☁️ Real-Time Sync Setup

To enable automatic sync across all your devices:

1. Go to console.firebase.google.com

2. Create a project → Realtime Database → Start in test mode

3. Project Settings → Your apps → Add Web App → Copy the config

4. Paste it at the top of this HTML file where it says YOUR_API_KEY

Setup Guide
Turn on your thermal printer and enable Bluetooth.
Pair the printer once via Android Settings → Bluetooth before using this app.
